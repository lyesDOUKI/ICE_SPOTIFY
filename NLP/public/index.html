Bienvenue sur l'API pour le NLP

Cette API permet d'analyser une requête et déduire l'intention de l'utilisateur en utilisant la bibliothéque spacy.

Comment utiliser l'API :

  1. Envoyez via la méthode GET un text à l'URL /nlp-process.
  2. L'API déduira l'intetion de l'utilisateur.
  3. L'API renverra le résultat action / objets sous forme de JSON.
